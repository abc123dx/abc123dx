## What I build

I build developer tools around AI infrastructure, mostly in TypeScript and
Python: clearer model interfaces, safer agent extensions, and more observable
agent runtimes.

专注于把 AI 基础设施做得更可组合、可审计、可观测。

## Working stack

`TypeScript` · `Python` · `Next.js` · `Node.js` · `OpenTelemetry` ·
`GitHub Actions`

## Current build direction

- Make model integrations easier to switch, inspect, and maintain.
- Turn Agent Skill and MCP security checks into deterministic, reviewable output.
- Turn AI agent telemetry into actionable performance, reliability, and cost
  summaries instead of opaque logs.
- Prefer local-first workflows, safe defaults, reproducible demos, and useful
  documentation.

## Build principles

```text
small interfaces  ·  observable behavior  ·  explicit trust boundaries
```

I prefer tools that remain understandable after the demo: typed boundaries,
documented trade-offs, and automation that can be reviewed in a pull request.
